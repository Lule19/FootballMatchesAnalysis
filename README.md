# FootballMatchesAnalysis
У овом пројекту потребно је коришћењем Apache Spark платформе извршити обраду скупа
података са свих одржаних утакмица фудбалских репрезентација од 1872. године до данас.
Дати скуп података можете скинути са адресе
https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017 и
користиће се фајлови results.csv и goalcorers.csv.
За захтевану обраду података можете користити програмски језик по избору, што значи
Java, Scala или Python. Потребно је приложити програмски код и дати резултат обраде за
следеће упите:
1. Са којом фудбалском репрезентацијом Србија (Serbia у datasetu) има најбољу, а са којом
најгору гол разлику. Овај задатак је потребно реализовати преко RDD колекције и mapreduce метода.
2. Који су све фудбалери постигли хет-трик (три поготка) на европским првенствима
(UEFA EURO) и на којим утакмицама. Формат колона:
3. Дати листу најбољих стрелаца, одн. прво и друго место по броју постигнутих голова, на
светским фудбалским првенствима (FIFA World Cup) од 1990. године па надаље.
Сортирати по 'Godina' растуће и 'Golova' опадајуће. Формат колона:
4. Дати 20 фудбалских репрезентација које су најдуже време биле без пораза после 1980.
године, навести у којем временском периоду, од кога су се десили порази, а укључити само
оне репрезентације које су некада учествовале на светским фудбалским првенствима.
Формат колона:
Oбратити пажњу да се мора извшити експлицитна конверзија колоне minute у тип int код
dataset-a goalcorers.csv преко:
goalscorersDF.withColumn("minute", $"minute".cast("int"))
